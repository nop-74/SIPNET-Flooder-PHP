🔅Браузерный SIP Флудер☎️

![Alt text](https://github.com/nop-74/SIPNET-Flooder-PHP/blob/main/SIP4.png)

- Программа работает через сип оператора - поставщика услуг связи. 
- Этот оператор, даёт после регистрации, 1 $ на счёт, и звонить можно бесплатно, 
- а потом просто сменить аккаунт. Или можно самому пополнять баланс и не менять аккаунт. 
- (Это для звонков по России) 
- А для звонков в разные страны, нужно, хотя бы один раз, заплатить оператору, 3 $ 

- Сайт оператора: www.sipnet.ru 

- Перед запуском программы, ты должен быть авторизован на сайте оператора. Сайт оператора, 
- можно не открывать в браузере, главное, что бы ты был там авторизован. Иногда, 
- проверяй авторизацию, она держится несколько дней. 
- И лучше, перед началом флуда, каждый раз делать повторную авторизацию на сайте, для проверки аккаунта. 

- Ну и бесплатный бакс получить не забудь у оператора, после регистрации. 
- В личном кабинете, вверху слева, нажми кнопку изменить статус, 
- привяжи телефон и получи бакс на счёт. 

- Можешь использовать их бесплатный бакс, который дают при регистрации, но это только для теста. 
- Для полноценного флуда, сип оператору, нужно хотя бы один раз, дать 3 $ 
- Лучший вариант, полноценного флуда - регистрировать аккаунт, нужно как юридическое лицо, а не частное. 

- Регистрация аккаунта, оформляется на юридическое лицо, по бесплатному телефону тех. поддержки оператора. 
- Для оформления требуют номер лицевого счёта. 

- Когда сделаешь акк на юр. лицо, тебе дадут отдельного менеджера. 
- Если будут вопросы, то сообщишь ему, что делаешь обзвон клиентов по старой базе телефонов.
